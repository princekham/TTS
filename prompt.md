# Installation
```
npm install
```
### Development Mode
```
npm run dev
```
### Production Mode
```
npm run build
npm run start
```

### To start Voxcmp
- set thee variable
- 
  ```
  set VOXCPM_USE_MODELSCOPE=1
  ```
- get to the environment first
  
  ```
  cd C:\Voxcpm\VoxCPM-main
  voxcpm-venv\Scripts\activate
  ```
  
```
python app.py --port 7860 --device auto
```
