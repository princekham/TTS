# Installation
```
npm install
```
### Development Mode
```
npm run dev
```
### Production Mode
```
npm run build
npm run start
```
### To start Voxcmp
```
cd C:\Voxcpm\VoxCPM-main
python app.py --port 7860 --device auto
```
